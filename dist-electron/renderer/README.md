# Assets

Esta pasta contém os recursos visuais do navegador.

## Ícones necessários

Para compilar o aplicativo, você precisará adicionar os seguintes ícones:

- `icon.png` - Ícone para Linux (PNG, 512x512)
- `icon.ico` - Ícone para Windows (ICO, múltiplos tamanhos)
- `icon.icns` - Ícone para macOS (ICNS)

Você pode criar ícones usando ferramentas como:
- [Electron Icon Maker](https://www.electron.build/icons)
- [Online ICO Converter](https://convertio.co/png-ico/)
- Ferramentas de design como GIMP, Photoshop, etc.

O ícone deve representar privacidade/segurança (sugestão: cadeado, escudo, etc.).
